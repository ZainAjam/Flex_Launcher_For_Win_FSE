# Flex_Launcher_For_Win_FSE
A launcher of typical windows gaming stores

Instructions:
1. Download the zip file to your desktop and unzip it
2. In the app_shortcut folder, change the shortcut locations to match the location of the app installs on your pc
3. In the assets > icons folder, change the icon files to match the app installs on your pc
4. OPTIONAL - Edit the config file to change the behaviour of the launcher or to add other apps to the launcher
